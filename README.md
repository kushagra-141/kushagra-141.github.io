# Kushagra Gupta — Portfolio

Personal portfolio website and resume, showcasing my work in Machine Learning, Computer Vision, and Data Engineering.

Live at: [https://kushagra-141.github.io/](https://kushagra-141.github.io/)

## 🚀 About Me
I'm a Computer Science Master's student at the Rochester Institute of Technology (RIT) focused on the intersection of machine learning, computer vision, and large-scale data engineering. 

- **Focus Areas**: Deep Learning, Computer Vision, Data Pipelines.
- **Experience**: 4 Internships (Vivriti Capital, Grant Thornton, etc.)
- **Research**: Co-authored IEEE Xplore paper on Parkinson's disease detection.

## 🛠️ Tech Stack
This website is built with a lightweight, high-performance stack:
- **Core**: React (via Babel-standalone for zero-build deployment)
- **Styling**: Tailwind CSS
- **Design**: Custom CSS for glassmorphism and animations
- **Deployment**: GitHub Pages

## 📁 Project Structure
- `index.html`: Main entry point (Portfolio)
- `Resume.html`: Interactive resume page
- `app.jsx`: React application logic
- `data.jsx`: Centralized portfolio content
- `sections/`: Modular React components for different portfolio sections
- `uploads/`: Static assets including PDF resume
